### 真结局：守夜人的传承 {#true}

<script>
  export default {
    onEnter({ globals }) {
      globals.flags.ending = "true";
      globals.crewMorale = 100;
    },
    view({ globals, locals }) {
      return {
        commanderName: locals.commanderName,
        boardedShip: !!(globals.flags || {}).boardedShip,
      };
    },
  };
</script>

你没有离开外星飞船。

守夜人的数据库被激活了。那些沉睡了两个世纪的系统重新开始运转——不是轰然启动，而是无声地、有序地、一层接一层地亮起来。墙壁上的半透明物质从暗蓝色变成了明亮的白色，舱室的温度开始上升，空气中出现了一种你从未闻过的气味——不像是灰尘或消毒剂，更像是春天雨后湿润的泥土。

数据库里储存的不只是"静默"的数据——是守夜人文明的整个知识体系。他们用了三千年的时间来研究和对抗"静默"，最终在覆灭之前把一切成果留在了这艘飞船里。他们知道自己会死。他们希望别人能活。

<span class="alien">你花了整整三天时间阅读守夜人留下来的记录。他们不是被"静默"消灭的——他们是耗尽了自己的文明资源，试图在"静默"到达别的文明之前找到对抗它的方法。他们成功了——至少理论上成功了。但他们来不及把方法传出去了。"静默"到达了他们的母星。他们只剩下了这艘船，和船上的最后一个船员。</span>

那个船员——名字已经无法辨认了，守夜人的名字是用某种复合频率表示的——在最后的时间里做了两件事：把自己的意识接入了通信系统以确保信号持续发送，并把自己的记忆存进了那个小球里。

{{#if boardedShip}}
你拿起那个小球。微光还在脉动。你把它接入了数据库——屏幕上出现了最后一条记录。是那个守夜人的声音——通过翻译器转换出来的声音异常平静。

"如果你看到这段记录——不要感到愧疚。我选择留在这里是我的选择。你来到这里——是你的选择。现在，你该做的不是为我哀悼。你该带着这些东西离开。带着我们三千年学到的一切。带着我们没能送给自己的未来。"
{{/if}}

你打开曙光号的通信频道。

"陈敏——把数据库里的所有内容打包，发送给联邦科学院的紧急频道。加密级别——最高。告诉他们——这不是威胁评估请求。这是解决方案。"

两个月后，联邦通过了"静默壁垒计划"。守夜人的技术被成功复现——一种能够在"静默"脉冲到达之前被激活的能量屏障，保护人类殖民星域内的每一个人。

守夜人把自己燃烧了两百年——只是为了给另一个文明点的一盏灯。

<span class="clue">他们做到了。你也做到了。</span>

### 好结局：曙光的使命 {#good}

<script>
  export default {
    onEnter({ globals }) {
      globals.flags.ending = "good";
      globals.crewMorale = Math.min(100, (globals.crewMorale || 80) + 20);
    },
    view({ globals, locals }) {
      return {
        commanderName: locals.commanderName,
        knowledge: globals.knowledge || 0,
        hasHighKnowledge: (globals.knowledge || 0) >= 3,
      };
    },
  };
</script>

你做出了一个清醒的决定——带着数据，撤回曙光号，返回联邦。

出发之前你最后看了一眼主屏幕上的外星飞船残骸——灰暗的、布满伤痕的外壳，一具靠在走廊墙上的遗骸，一缕持续了两百年的蓝光。你把那个画面保存了下来。

跃迁引擎工作了。

曙光号用了十七天回到了联邦星域。你立即联系了联邦科学院，把曙光号收集到的所有数据——信号记录、探测球影像、外星飞船的能量分析和壳体外壳的材料样本——全部交付给了他们。

{{#if hasHighKnowledge}}
科学家们花了三个月完成了初步分析。守夜人的技术——至少可用于对抗"静默"的那部分——最终被成功复现。第一批能量屏障在人类殖民星域的边界被激活的那天，你把曙光号船员召集到了舰桥上。

"我们是第一批见到守夜人信号的人类，"你对大家说，"也许有一天——会有人类站在另一个文明的飞船里，和另一个即将消亡的文明做同样的事。希望到那一天——他们能想起守夜人。"
{{else}}
科学家们花了整整一年才完成对数据的初步整理。时间紧张——但幸好"静默"的脉冲在最后阶段有一段时间的减速，给了人类多出了一段窗口期。能量屏障最终赶在脉冲到达之前完工了。

事后联邦给你颁发了一枚勋章。你把勋章放在曙光号的舰桥里，和那个外星飞船残骸的照片放在一起——不是为了纪念你，是为了纪念两百年前死在那艘船上的那个不知道名字的人。
{{/if}}

曙光号的使命完成了。你的使命也完成了。

### 坏结局：静默降临 {#bad}

<script>
  export default {
    onEnter({ globals }) {
      globals.flags.ending = "bad";
      globals.crewMorale = 0;
    },
    view({ globals, locals }) {
      return {
        commanderName: locals.commanderName,
      };
    },
  };
</script>

你做了一个任何人都不应该做的决定。

曙光号的武器系统——设计用来对抗小行星和太空碎片的——被启动并瞄准了"静默"脉冲。你的逻辑很简单：用能量对抗能量。

罗伊在最后一秒还在警告你："指挥官——'静默'不是能量脉冲！它是一种——我们甚至不完全理解它是什么！对它开火——可能会把它激活，也可能会把我们——"

你没有听。命令下达了。

导弹发射了。它击中了"静默"脉冲的前沿——然后，一切都变了。

不是爆炸。是沉默。

曙光号的每一个系统、每一个屏幕、每一个指示灯，在同一个瞬间熄灭了。舰桥陷入了一片绝对的黑暗——不是普通的黑夜，是那种连记忆中的光都无法想象的虚无。你伸手去摸自己的控制台——你的手在动，但你感觉不到自己的手指了。你没有听到任何人的声音——不是因为他们没有说话，是因为你听不到了。

然后你的意识——你最私密的那一部分，你自己就是自己、每天早晨醒来时确认的那一部分——它开始散开。不是疼痛——是一种缓慢的、无法描述的稀释。像是在水中扩散的一滴墨水。你的记忆、你的姓名、你的恐惧、你的责任、你关于曙光号的一切——不再属于你了。它们一片一片地飘进了黑暗中，而曾经是"你"的那个东西——正在变得空无一物。

守夜人的信号停止播出了。人类星域通信网络同时瘫痪。联邦没有收到任何警告——也不可能收到任何警告了。

两百年前那艘船上的最后一个船员，用尽了自己的一切——为了人类不被赶上这一幕。但有时候，不是警告不够响亮，是听的人不相信自己听不懂的话。

<span class="warning">静默不需要杀死你。它只需要让你忘记自己是谁。</span>
